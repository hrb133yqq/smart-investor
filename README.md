# smart-investor

1. 收集市场数据: 主要为报表数据。可以是html格式或pdf格式。
2. 数据萃取与更正: 生成统一json格式的数据。
3. 数据加工: 数据单位标准化。
4. 原则应用：展示符合一定原则的股票数据与指标。
